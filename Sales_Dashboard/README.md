# Sales Dashboard using Power BI

## Project Overview

This dashboard analyzes sales performance across different regions, products, and customer segments.

## Features

- KPI Cards
- Sales Trend
- Profit Analysis
- Region-wise Sales
- Category Analysis
- Interactive Filters

## Tools Used

- Power BI
- Excel
- DAX
- Power Query
